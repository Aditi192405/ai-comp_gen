# 🚀 AI Component Generator

AI Component Generator is a React-based web application that uses Google's Gemini AI to generate modern, responsive, and animated UI components from simple text prompts.

## ✨ Features

- 🤖 AI-powered UI generation using Gemini AI
- 🎨 Generate modern and responsive designs
- ⚡ Support for:
  - HTML + CSS
  - HTML + Tailwind CSS
  - HTML + Bootstrap
  - HTML + CSS + JavaScript
- 📝 Built-in Monaco Code Editor
- 📋 One-click code copy
- 📥 Download generated code as HTML file
- 🌐 Live preview support
- 📱 Fully responsive interface

## 🛠️ Tech Stack

- React.js
- Vite
- Google Gemini API
- Monaco Editor
- React Router
- React Select
- React Icons
- React Toastify
- React Spinners
